VSCode使用技巧

发表于 2024-01-15 分类于 技术

VSCode使用技巧（Window）

命令

  • 先决条件：安装VSCode软件时将其加入环境变量PATH中
  • 命令：code 选项 目录/文件路径
    • 常用命令
      • code -h：获取code命令帮助
      • code -r path：窗口复用（关闭当前窗口，新建窗口打开指定文件）
      • code -r -g path:number：窗口复用并定位到指定文件的指定行（可用于当代码报错时定位到指定位置）
      • code -r -d file1path file2path：窗口复用并对比两个文件内容
    • 与管道联用：ls | code 选项 目录/文件路径
    • 扩展管理（Extensions management）
    • 故障排除（Troubleshooting）

文本操作（包括代码）

  • 光标移动
    • 单个字符移动（移动单个字符）：左/右方向键
    • 单个单词移动（移动单个单词）：Ctrl+左/右方向键
    • 行内移动（移动到行首/尾）：HOME/END
    • 多行移动：上/下方向键
    • 代码块移动（通过匹配括号进行移动）：Ctrl+Shift+\
    • 整个文档移动（移动到文档首/尾）：Ctrl+HOME/END
    • 撤销光标移动（移动光标后希望光标退回到原来位置）：Ctrl+U
  • 文本选择
    • 单个字符选择：Shift+上/下/左/右方向键 或者 鼠标左键单击当前光标所在字符
    • 单个单词选择：Ctrl+Shift+左/右方向键 或者 鼠标左键双击当前光标所在单词
    • 单行选择：Ctrl+HOME/END 或者 鼠标左键三击当前光标所在行 或者 鼠标左键单击文本左侧行号
    • 多行选择：Shift+上/下方向键 或者 Ctrl+Shift+HOME/END 或者 鼠标左键单击文本左侧行号并向上或向下拖动选中指定文本
    • 全选：Ctrl+A 或者 鼠标左键四击
  • 文本移动
    • 普通方式：选中文本 剪切文本 粘贴文本
    • 单行移动：Alt+上/下方向键
    • 多行移动：选中指定的文本后 使用ALT+上/下方向键
    • 鼠标操作：选中指定文本后 拖动到指定位置
  • 文本复制
    • 普通方式：选中文本 复制文本 粘贴文本
    • 单行复制：Alt+Shift+上/下方向键
    • 多行复制：选中指定文本后 使用Alt+Shift+上/下方向键
    • 鼠标操作：选中指定文本后 按住Ctrl键后拖动文本到指定位置
  • 文本删除
    • 普通方式：选中文本后按下回车或者delete键
    • 文本行删除：Ctrl+Shift+K（删除文本） 或者 Ctrl+X（剪切文本）
  • 换行符
    • 普通换行：Enter
    • 当行下添加换行：Ctrl+Enter
    • 当行上添加换行：Ctrl+Shift+Enter
  • 缩进
    • 单行缩进：Ctrl+[/]
    • 多行缩进：选中指定文本后 使用 Ctrl+[/]
  • 制表符
    • 正向制表符：tab
    • 反向制表符：Shift+tab

默认快捷键

  • 打卡设置界面：Ctrl+,
  • 活动栏
    • Ctrl+Shift+E：文件资源管理
    • Ctrl+Shift+F：跨文件搜索
    • Ctrl+Shift+G：源代码管理
    • Ctrl+Shift+D：启用和调试
    • Ctrl+Shift+X：扩展管理
  • 命令面板
    • F1/Ctrl+Shift+P：命令面板
  • 底部状态栏
    • Ctrl+Shift+M：底部面板（默认位置，可调整）-问题
    • Ctrl+Shift+U：底部面板（默认位置，可调整）-输出
    • Ctrl+Shift+Y：底部面板（默认位置，可调整）-调试控制台
    • Ctrl+`：底部面板（默认位置，可调整）-终端
  • 编辑器
    • Ctrl++：放大编辑器界面（包括字体图标等）
    • Ctrl+-：缩小编辑器界面（包括字体图标等）
  • 代码
    • 代码格式化：Alt+Shift+F
    • 代码注释：Ctrl+/
    • 代码自动补全
      • 补全代码：输入关键字后弹出自动补全窗口可选择想要的函数后按下tab自动补全代码
      • 隐藏代码补全窗口：Escape
      • 隐藏代码不全窗口后显示窗口：Ctrl+空格键
    • 快速修复建议列表

光标

  • 多光标
    • 使用鼠标：按住鼠标中键向上/下拖动
    • 使用键盘
      • 当前光标位置垂直向上或向下创建多光标：Ctrl+Alt+上/下方向键
      • 匹配相同内容并在其后创建光标：Ctrl+D
    • 鼠标+键盘：Alt+鼠标左键单击指定位置

自定义快捷键

命令面板

  • 行排序
    • 按升序排列行：sort lines Ascending
    • 按降序排列行：sort lines Descending
  • 合并行：join lines
  • 转换为大/小写
    • 转换为大写：transform to uppercase
    • 转换为小写：transform to lowercase
  • 转置光标处字符：transpose characters around the cursor
  • 缩进：indent

用户配置文件setting.json

  • 代码补全窗口：”editor.quickSuggestions”
  • 参数预览窗口：”editor.parameterHints”
